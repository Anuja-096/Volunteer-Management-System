Volunteer Management System

Overview

The Volunteer Management System is a Python-based application developed to help NGOs manage volunteer records efficiently.

Features

Add Volunteer
View Volunteer Records
Search Volunteer
Update Volunteer Details
Delete Volunteer
Generate Volunteer Reports
Display Total Volunteers

Technologies Used

Python
File Handling

Project Purpose

This project was developed to simplify volunteer data management for organizations and NGOs. It allows easy storage, retrieval, updating, and reporting of volunteer information.
